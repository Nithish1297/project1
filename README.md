# 📊 Retail Sales & Customer Segmentation Analysis

## 📌 Objective
Analyze retail sales data to identify customer behavior, revenue trends, and key business insights.

---

## 🛠️ Tools Used
- Python (Pandas, Matplotlib)
- SQL (MySQL)
- Power BI

---

## 📥 Dataset
https://www.kaggle.com/datasets/vijayuv/onlineretail

---

## 🔍 Project Steps
1. Data Cleaning using Python
2. Exploratory Data Analysis (EDA)
3. Customer Segmentation (RFM)
4. SQL Business Queries
5. Power BI Dashboard

---

## 📊 Key Insights
- Top 10% customers generate ~60% revenue
- Sales peak during November–December
- UK dominates total revenue
- Few products drive majority sales
